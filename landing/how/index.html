How does Project Colony Work

Create a project

See your current projects on the project page.

Add Project

You can simply add a project by clicking ‘Add Project’.

The project creates a new colony on the etherium blockchain.

Project Details

Double click on a project to see its details.

In the project details you can change the project name and the project description. On top you can see and add tasks to your project.

Add Task

By clicking on ‘Add’ you can add a task.

You can enter the - Task Name - Task Description - Collaborators email address - The hash of the contract document

By clicking ‘save’ the task will be persited in your project/colony on the blockhain with the contract hash persisted.
